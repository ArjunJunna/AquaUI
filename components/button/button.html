Button

Buttons allow users to take actions, and make choices, with a single click.

Example for primary button.

Primary buttons are used whenever your user click on a link or button. 
    
         <button class="button btn-primary">Primary Button</button>
         <button class="button btn-primary btn-outline">Outline Button</button>

Example for link button.

Link buttons are used for a link action.
Link Button 
    
          <a href="#" class="btn btn-primary btn--link">Link Button</a>

Example for icon button.

Icon buttons are used in cases when you need an icon to act as buttons for users. 
    
          <span class="btn-icon"><i class="fas fa-home"></i></span>
          <span class="btn-icon"><i class="fas fa-star"></i></span>
          <span class="btn-icon"><i class="fas fa-plus"></i></span>

Example for floating action button.

Floating-action buttons can be customized by adding the font-awesome icon of your choice. 
    
          <span class="btn-icon" id=""><i class="fas fa-camera"></i></span>
          <span class="btn-icon" id=""><i class="fas fa-plus"></i></span>
          <span class="btn-icon" id=""><i class="fas fa-arrow-up"></i></span>
